# PointFlow

PointFlow is a .NET app for managing tasks and rewards through a point-based system. Tasks earn points, rewards cost points, and rewards stay locked until the required balance is available.


## Planned Features

- Point-based tasks and rewards
- Reward locking based on available points
- Timers for both tasks and rewards
- Optional Pomodoro mode for tasks
- Session-by-session learning checkpoints in Pomodoro mode
- LLM-generated quizzes between Pomodoro sessions to unlock the next session

## Concept

The goal is to combine productivity tracking, delayed gratification, and structured learning into one workflow. A user earns points by completing tasks, spends points on rewards, and can use timed task sessions to stay focused. In Pomodoro mode, each completed study session requires recording what was learned and passing a quiz before continuing.
